# Mealmory: 식사의 추억

내가 섭취한 음식을 검색해 탄단지 함유량을 확인하고, 검색 결과를 토대로 식단을 저장하는 서비스입니다.

## skills
- 코어
  - typescript
  - next

- 스타일링
  - tailwind css

- 상태관리
  - zustand

- 통신
  - return-fetch

- 배포
  - vercel

## library
- 식단 indexing : uuid
- alert 인터렉션 : sweet-alert-2
- 통계 데이터 시각화 : react-chartjs-2
- 다크모드 테마  : tailwind css, next theme
